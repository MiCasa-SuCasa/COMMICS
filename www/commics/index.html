ABOUT COMMICS

Commics is a web tool for COMparative metagenoMICS. It provides access to publicly available metagenomes and allows the functional comparison among them.
The submission of custom data is also allowed, the user can upload the functional annotations of in-house metagenomics sequencing projects.
Commics offers three different aproaches to the functional comparision
  • Two-way clustering: Allows to cluster those metagenomes that are more similar, according to their functional profile
  • Functional profile: Shows the relative abundance of different functional categories present in a metagenome
  • Canonical discriminant analysis: Test the hypothesis that different environments have different metabolic profile
